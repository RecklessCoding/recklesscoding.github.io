Projects Suggestions

Rainforcement Learning for Reactive Planning

Cognitive Architectures, Reactive Planning, Machine Learning, Systems AI

This project will investigate how we can use reactive learning in a reactive planning paradigm, e.g. BOD, to optimise the plan structure.

Using the GB to establish constraints in real-time decision making

Validation and Verification, Transparency, Systems AI

In Aler Tubella et al. (2019), we proposed the development of the Glass Box (GB) for validating decision making. The GB can be used to inform the action-selection system of an agent if an action is permissible or not given the normative moral constraints set by the user. However, in highly complex environments, conflict between moral values may arise. This ambitious project explores how we can implement and intergrate the GB into agents.

Graphical display of hierarchical norms

Responsible AI, cognitive science

Using Value-Sensitive Design, one can break high-level abstract socio-ethical values into concrete norms by producing a hiearchy of norms. This undergraduate project looks into such a hierarchy can be effectively visualised. Ambitious students can look into extending this work by introducing provisions for real-time transparency.

Cognitive models of moral decision making

Responsible AI, cognitive science, experimental philosophy

This work is a continuation of Wilson & Theodorou 2019. It will involve the development of a VR simulator and running of relevant experiments.
