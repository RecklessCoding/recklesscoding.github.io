## What is project?
This is my personal website. After getting fed up by having to enter the same information to 2-3 different systems/formats, I decided to move to BibTeX-only solutions. This gave me an opportunity to redo my whole website. Unlike other papes and templates found, this one is just HTML and Javascript. Everything runs client side, significantly reducing both its complexity and requirements for hosting (no PHP installation is needed!).

## Can I use your code/template for my own website?
Absolutely! If you do, make sure you include the MIT License. It will also be appreciated if drop me an email/message here to let me know. This way, with your agreement, I can list your page in this section. 

## Technologies used
This template is implemented in [Bootstrap](https://getbootstrap.com/); a popular HTML, CSS, and JS framework for developing responsive websites that scale depending the type of your device and size of your screen. While the project started with the *Flatly* theme, it has slowly moved towards its own colouring scheme over a number of CSS files.

Most importantly, this project uses a modified version of the [BibTeX-js](https://github.com/pcooksey/bibtex-js) library to parse BibTeX files and render their items as HTML elements. This parser is used to popularise the *Publications* and *Outreach* pages.
